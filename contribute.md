# 为仓库贡献

## 贡献指南
1. 下载并安装 [Node.js](https://nodejs.org/) 
2. Fork 本仓库并克隆到本地。
3. 创建一个新的分支 `git checkout -b feature/your-feature-name` 或 `git checkout -b bugfix/your-bugfix-name`。
4. (可选)解包最新的精灵数据包到 `public\data\BinData` 目录下，覆盖原有数据。
5. 用 vs code 打开项目，在终端运行 `yarn` 安装依赖。
6. 运行 `yarn sync:pet-data` 同步精灵数据。
7. 运行 `yarn dev` 启动开发服务器.
